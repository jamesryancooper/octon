# Ideas

Brainstorming space for possibilities, "what if" explorations, and unfiltered thinking.

## Suggested Organization

- One file per topic or exploration
- Use descriptive filenames: `auth-approach.md`, `ui-redesign.md`
- Date-prefix for time-sensitive ideas: `2025-01-04-quick-thought.md`

## When to Promote

When an idea becomes a decision or next action, promote it using `workflows/promote-from-scratch.md`.
