# Research

Collected findings, analysis, and reference materials.

## Suggested Organization

- One file per research topic: `auth-options.md`, `performance-benchmarks.md`
- Use subdirectories for large investigations: `research/auth/`
- Include sources and links within each file

## When to Promote

When findings lead to decisions or constraints, promote distilled summaries using `workflows/promote-from-scratch.md`.
