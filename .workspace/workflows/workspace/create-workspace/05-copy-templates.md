# Step 5: Copy Template Structure

```text
.workspace/templates/workspace/    →    <target>/.workspace/
├── START.md                            ├── START.md
├── scope.md                            ├── scope.md
├── conventions.md                      ├── conventions.md
├── progress/                           ├── progress/
│   ├── log.md                          │   ├── log.md
│   └── tasks.json                      │   └── tasks.json
└── checklists/                         └── checklists/
    └── complete.md                             └── complete.md
```
