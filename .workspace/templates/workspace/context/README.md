# Context

Background knowledge that agents need to work effectively in this workspace.

## Required Files

| File | Purpose |
|------|---------|
| `decisions.md` | Agent-readable decision summaries |
| `lessons.md` | Anti-patterns and failures to avoid |
| `glossary.md` | Domain-specific terminology |
| `dependencies.md` | External systems and references |
| `constraints.md` | Technical and business rules |

## Optional Files

Add these based on workspace needs:

| File | When to Add |
|------|-------------|
| `compaction.md` | Workspaces with long sessions or complex multi-step work |
| `tools.md` | Workspaces where agents use specialized tools or need tool selection guidance |

See `docs/architecture/workspaces/context.md` for format and examples.
