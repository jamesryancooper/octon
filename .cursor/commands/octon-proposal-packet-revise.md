# Proposal Packet Revise

Run the `revise-proposal-packet` bundle for one packet path or proposal id.

Apply only packet-local revisions needed to address review findings, write
`support/revisions/<revision-id>.md`, refresh packet catalog, checksums, and
registry projections as needed, and keep or return the packet to `in-review`.

Do not mark the packet accepted from this command; route back to
`review-proposal-packet` after revision.
