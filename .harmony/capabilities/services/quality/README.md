# Quality

Test, evaluate, and benchmark.

## Services

- [Test](./test/guide.md) — UI/E2E (Playwright), API contracts (Pact), fuzz tests (Schemathesis); emits reports and status checks.
- [Eval](./eval/guide.md) — Verifies docs, code, and diffs; checks contracts, links, style, hallucination; emits pass/fail gates.
- [Bench](./bench/guide.md) — Performance baselines and PR regression checks.
- [A11y](./a11y/guide.md) — Accessibility checks (WCAG compliance).
- [Seed](./seed/guide.md) — Test fixtures and data seeding.
