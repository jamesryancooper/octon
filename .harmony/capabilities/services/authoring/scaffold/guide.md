# Scaffold — Project & Feature Bootstrapping

- **Purpose:** Generate services/features/CI from Stack profiles.
- **Responsibilities:** templates, wiring, boilerplate tests/docs.
- **Integrates with:** Stack (source of truth), Prompt (templating), Tool (git/CI).
- **I/O:** new folders, files, PRs.
- **Wins:** From decision to code quickly.
- **Harmony default:** Monolith-first template with **Turborepo** caching, **OpenTelemetry** hooks, 12-Factor envs, and preview-ready **Vercel** deploy.
