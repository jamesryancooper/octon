# Checkpoints

Record these checkpoints while auditing:

1. Canonical statement extraction complete
2. Traceability map complete
3. Normative clause inventory complete
4. Authority/accountability map complete
5. Conflict and gap log complete
6. Rewrite pack complete
7. Final scores complete

If a checkpoint cannot be completed from the charter alone, record that as a finding rather than consulting outside materials.
