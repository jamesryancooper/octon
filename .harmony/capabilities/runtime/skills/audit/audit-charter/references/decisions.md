# Decision Rules

## Closed-Book Rule

If a fact is not stated in the charter, treat it as absent for audit purposes.

## Conflict Classification

- `Direct contradiction`: two statements cannot both govern the same situation as written.
- `Latent conflict`: wording allows multiple plausible interpretations or unbounded override paths.
- `Gap`: a required owner, threshold, definition, or control path is missing.

## Severity Guidance

- `High`: breaks standalone operability, authority resolution, or core enforcement.
- `Medium`: weakens determinism, testability, or accountability but does not fully break the charter.
- `Low`: wording or framing issue with limited operational risk.

## Rewrite Rule

Only propose rewrites for issues that materially affect clarity, alignment, enforceability, or auditability.
