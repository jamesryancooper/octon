# Glossary

- **Cross-subsystem contract:** Any declaration in one subsystem consumed by another.
- **Coherence edge:** Reference or dependency link between subsystem artifacts.
- **Ownership collision:** Conflicting claims over authority for the same contract surface.
- **Policy conflict:** Incompatible guidance between governing documents.
