# Interaction

Ask the user only when one of these materially changes the plan:

- target score expectations,
- whether scope may expand beyond the charter,
- whether broader contract or schema updates are allowed in follow-on work,
- unresolved tie-breaks in findings severity or governance profile.

If the user does not answer and the risk is reversible, proceed with the narrowest reasonable assumption and record it explicitly.
