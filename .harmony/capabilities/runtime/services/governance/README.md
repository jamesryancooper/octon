# Governance

Enforce policy, compliance, and security.

## Services

- [Policy](./policy/guide.md) — Codifies repo and delivery rules (licenses, versions, paths, CODEOWNERS); enforces via OPA.
- [Guard](./guard/guide.md) — Redacts PII/secrets, validates tool calls, enforces path and egress allowlists.
- [Compliance](./compliance/guide.md) — Maps OWASP ASVS v5 and NIST SSDF controls to tasks; produces auditable evidence bundles.
- [Headers](./headers/guide.md) — Security headers policy enforcement.
- [Vault](./vault/guide.md) — Secrets wrappers; scanning and rotation hooks.
