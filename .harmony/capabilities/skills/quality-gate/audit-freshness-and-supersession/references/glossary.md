# Glossary

- **Freshness anchor:** Source artifact whose changes should trigger downstream updates.
- **Supersession chain:** Linked sequence showing which artifact replaced another.
- **Current-state marker:** Signal that an artifact is considered authoritative now.
- **Orphan historical artifact:** Old artifact with no valid link to its replacement.
