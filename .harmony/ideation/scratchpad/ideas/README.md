# Ideas

Quick captures for possibilities, "what if" explorations, and unfiltered thinking.

## Purpose

This is the **lowest friction** capture space. Jot down thoughts without worrying about structure or whether they're worth pursuing.

## Suggested Organization

- One file per topic or exploration
- Use descriptive filenames: `auth-approach.md`, `ui-redesign.md`
- Date-prefix for time-sensitive ideas: `2025-01-04-quick-thought.md`
- Keep entries brief — if you're writing paragraphs, consider `brainstorm/`

## The Funnel

```
ideas/  →  brainstorm/  →  projects/
(here)     (explore)       (commit)
```

Most ideas stay here and eventually get deleted. That's fine — the value is in capture, not retention.

## When to Graduate

Move to `brainstorm/` when:
- The idea is worth structured exploration
- You're spending more than a few minutes on it
- You want to track questions and validate before committing

Move directly to `projects/` when:
- You're already confident it's worth pursuing
- It will span multiple sessions
- It will produce workspace artifacts
