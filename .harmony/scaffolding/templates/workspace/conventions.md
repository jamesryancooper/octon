# Conventions

## File Naming

- Primary convention: {{NAMING_CONVENTION or "lowercase with hyphens: `my-file.md`"}}
- Prompts: `{action}-{target}.md`
- Workflows: `{verb}-{noun}.md`

## Document Structure

```markdown
# Title (action-oriented)

## Context (1-2 sentences max)

## Instructions (numbered list)

## Output (what to produce)
```

## Writing Style

| Do | Don't |
|----|-------|
| Use imperative verbs | Explain why (save for `ideation/scratchpad/` or `docs/`) |
| Use lists over prose | Write paragraphs |
| Be specific and concrete | Use vague language |

## Progress Log Format

```markdown
## YYYY-MM-DD

**Session focus:** [one-line summary]

**Completed:**
- [task 1]

**Next:**
- [priority item]

**Blockers:**
- [if any]
```
