# Decisions

- Use the pack-local prompt assets as the runtime source of prompt truth.
- Preserve the bundle family under one stable dispatcher with
  `source-to-architecture-packet` as the default route.
- Keep ordinary architecture revision and constitutional challenge as explicit
  leaf bundles rather than overloading the default source-to-architecture flow.
- Keep the alignment audit and executable-prompt generator as companions, not
  numbered stages.
- Prefer the composite dispatcher skill as the reusable core and the command
  wrapper as the stable family gateway.
