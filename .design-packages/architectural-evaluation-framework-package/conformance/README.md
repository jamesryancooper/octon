# Conformance

Use this module for machine-checked behavioral proof or scenario validation.

## Scenario Suites

- `scope-classification`
  - verifies supported and unsupported target classification
- `mode-selection`
  - verifies whole-harness vs bounded-domain selection

## Draft-Status Rule

Draft packages may keep a small scenario set while the package is still being
authored. The validator must still exist and validate scenario shape whenever
scenarios are added.
