# Checkpoints

- Change identity resolved.
- Route selected from canonical policy.
- Lifecycle outcome selected separately from route.
- Touched-path scope reviewed.
- Validation floor satisfied or blocker recorded.
- Review evidence or waiver recorded when required.
- Durable history reference recorded.
- Integration, publication, and cleanup statuses recorded.
- Rollback handle recorded.
- Change receipt written or blocker preserved.
