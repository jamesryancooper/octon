# Glossary

- **Docs-as-code:** Documentation artifacts versioned and reviewed with source.
- **Coverage proof:** Evidence of both findings and checked-clean files.
- **Canonical guidance:** Approved policy/template source used for validation.
- **Template drift:** Documentation structure diverges from canonical template.
- **Remediation batch:** Grouped set of fixes ordered by severity.
