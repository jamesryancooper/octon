# Glossary Reference

**Required when capability:** `domain-specialized`

## Core Terms

| Term | Definition | Example |
|------|------------|---------|
| preview deployment | Non-production deployment for validation | PR-linked temporary URL |
| production deployment | Live deployment serving users | Stable project domain target |
| vercel link | CLI command binding local repo to Vercel project | Project metadata bootstrap |
| build output | Compiled artifact uploaded by Vercel CLI | Framework build directory |
| project settings | Runtime/build/env configuration in Vercel | Framework preset and env vars |
