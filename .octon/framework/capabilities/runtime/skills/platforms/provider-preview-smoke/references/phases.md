# Behavior Phases

## Phase 1: Pre-flight

- Resolve requested provider context and target surface.
- Verify required inputs and non-destructive execution assumptions.

## Phase 2: Provider Mapping

- Collect provider-specific signals relevant to the requested task.
- Map provider behavior to canonical Octon tier/ACP expectations.

## Phase 3: Evidence and Output

- Produce concise, auditable evidence for the task.
- Record run summary and escalation notes when applicable.
