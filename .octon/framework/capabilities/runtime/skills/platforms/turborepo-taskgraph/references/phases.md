# Behavior Phases

## Phase 1: Pre-flight

- Confirm request scope and required inputs.
- Validate non-destructive execution assumptions.

## Phase 2: Analysis and Mapping

- Gather evidence from the requested surfaces.
- Map findings to canonical Octon contracts.

## Phase 3: Output and Escalation

- Emit a concise, auditable result with explicit evidence.
- Provide escalation guidance when confidence or context is insufficient.
