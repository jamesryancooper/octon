# Glossary Reference

**Required when capability:** `domain-specialized`

## Core Terms

| Term | Definition | Example |
|------|------------|---------|
| composition | Building UI by combining small components | Compound component APIs |
| SSR | Server-side rendering | First paint HTML generated on server |
| hydration | Client attaching behavior to server-rendered HTML | React boot on initial page load |
| prop drilling | Passing props through multiple layers | Repeated intermediary component props |
| variant | Explicit component mode | Button `size` and `tone` variants |
