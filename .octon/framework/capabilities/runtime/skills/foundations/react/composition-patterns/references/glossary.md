# Glossary Reference

**Required when capability:** `domain-specialized`

## Core Terms

| Term | Definition | Example |
|------|------------|---------|
| compound component | Coordinated component family sharing context | `Tabs`, `Tabs.List`, `Tabs.Panel` |
| state lifting | Moving shared state up to common parent | Synchronizing sibling components |
| explicit variant | Named mode instead of implicit boolean behavior | `variant="primary"` |
| slot API | Structured subcomponent insertion points | Header/body/footer composition |
| controlled component | Parent owns state via props/events | Form input with external state owner |
