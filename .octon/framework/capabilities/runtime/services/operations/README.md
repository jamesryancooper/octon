# Operations

Runtime infrastructure, scheduling, and observability.

## Services

- [Tool](./tool/guide.md) — Unified, sandboxed action wrappers (shell, git, HTTP, file I/O, Vercel CLI).
- [Schedule](./schedule/guide.md) — Cron-like scheduling for plans and runs (daily/weekly jobs, delta crawls).
- [Cost](./cost/guide.md) — LLM cost management; token tracking, budget enforcement, model selection by tier.
- [Cache](./cache/guide.md) — Memoization and artifact caching; content-addressable storage with TTLs.
- [Observe](./observe/guide.md) — OpenTelemetry traces, metrics, logs; JSONL traces, redaction markers, run records.
