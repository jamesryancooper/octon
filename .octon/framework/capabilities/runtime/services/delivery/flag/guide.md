# Flag — Feature Flags & Progressive Delivery

- **Purpose:** Manage feature toggles, kill switches, and rollout policies.
- **Responsibilities:** environment‑scoped flags, segmentation, experiment toggles, emergency off.
- **Integrates with:** Release/Patch (flag-aware notes), Stack (defaults), Policy (approval rules).
- **I/O:** flag configs, rollout events.
- **Wins:** Lower change-fail rate and safe rollouts.
- **Octon default:** Backed by Vercel Edge Config for instant, atomic flag changes.
