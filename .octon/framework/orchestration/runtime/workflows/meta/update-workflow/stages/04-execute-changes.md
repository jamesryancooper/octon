# Step 4: Execute Workflow Changes

## Purpose

Apply changes to the canonical workflow unit, then regenerate `guide/` instead
of editing generated README files by hand.

## Actions

1. Update `workflow.yml`.
2. Update canonical stage assets.
3. Update workflow manifest and registry metadata when needed.
4. Regenerate `guide/` with the guide generator.

## Output

- Updated canonical workflow contract
- Updated stage assets
- Regenerated README
