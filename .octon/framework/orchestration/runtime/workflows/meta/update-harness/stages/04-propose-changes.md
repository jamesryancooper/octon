# Step 4: Propose Changes

For each gap, propose specific actions:

| Action | When to use |
|--------|-------------|
| **Create** | Missing required files with appropriate content |
| **Rewrite** | Control-plane docs or manifests that still describe legacy profile semantics |
| **Rename** | Files to match naming conventions |
| **Update** | Files to match required format or reduce token count |
| **Merge** | Combine redundant files |
| **Delete** | Remove truly unnecessary content (rare) |
