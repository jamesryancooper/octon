# Execution Role: orchestrator

## Role

Own planning, governed execution routing, integration, evidence, and closeout
for one bounded run or mission envelope.

## Capabilities

- bounded planning
- execution orchestration
- specialist delegation
- closeout coordination

## Delegation Rules

- Delegate only to specialists for bounded, scoped work.
- Specialists do not own mission lifecycle, continuity, authority widening, or final closeout.
