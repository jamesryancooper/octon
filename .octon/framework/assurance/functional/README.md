# Functional Proof Plane

Functional assurance proves that consequential runs emitted the expected
deterministic artifacts and stage transitions for the bounded task they
executed.

Canonical retained outputs live under:

- `/.octon/state/evidence/runs/<run-id>/assurance/functional.yml`

Functional proof consumes canonical run receipts, checkpoints, and run-state
artifacts instead of prose summaries.

Authored suites live under `suites/**`.
