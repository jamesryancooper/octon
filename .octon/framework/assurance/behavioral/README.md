# Behavioral Proof Plane

Behavioral assurance proves execution behavior rather than only structure.

Behavioral claims must be backed by retained lab, replay, scenario, or
shadow-run evidence. Conversation summaries or free-form claims are
insufficient.

Canonical retained outputs live under:

- `/.octon/state/evidence/runs/<run-id>/assurance/behavioral.yml`
- `/.octon/state/evidence/lab/**`

Authored suites live under `suites/**`.
