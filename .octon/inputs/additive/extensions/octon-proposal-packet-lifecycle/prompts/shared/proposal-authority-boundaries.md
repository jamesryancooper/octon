# Proposal Authority Boundaries

Generated prompts, generated support files, manual prompt variants, source
context, proposal packets, generated proposal registry projections, GitHub
comments, PR labels, CI dashboards, chat history, model memory, tool
availability, browser state, external dashboards, and external workflow engines
do not become Octon authority, control truth, runtime policy, or permission.

Durable authority may land only in declared promotion targets outside active
proposal paths and only after validation, publication, and retained evidence
support the claim.
