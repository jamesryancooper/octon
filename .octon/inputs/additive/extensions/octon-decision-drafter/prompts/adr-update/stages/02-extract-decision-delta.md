# Extract Decision Delta

Extract the decision-level change from the diff and retained evidence.

## Produce

- a concise summary of what changed materially
- the affected ADR sections, if any
- the evidence-backed rationale for the change
- explicit unknowns, caveats, or missing proof

## Rules

- distinguish implementation detail from decision-level intent
- cite only facts supported by the diff and retained evidence
- if the diff does not justify an ADR-level update, say so plainly
