# Rollback Notes Bundle Contract

This bundle drafts non-authoritative rollback notes from a diff plus rollback
posture or run context and retained evidence.

## Required Outcome

- normalize rollback-posture or run-contract context
- derive reversibility, trigger, and contamination implications
- produce one reviewable rollback-notes section

## Additional Guardrails

- never auto-edit rollback posture truth
- never present a guessed recovery path as canonical execution control
- stop when the available evidence does not support rollback-specific claims
