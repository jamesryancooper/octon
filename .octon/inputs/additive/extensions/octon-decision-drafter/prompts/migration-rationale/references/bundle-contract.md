# Migration Rationale Bundle Contract

This bundle drafts a non-authoritative migration-rationale section from a diff
plus migration context and retained evidence.

## Required Outcome

- normalize migration-plan or proposal-packet context when present
- synthesize compatibility, cutover, rollback, and unknowns
- produce one reviewable rationale draft

## Additional Guardrails

- never auto-edit `migrations/index.yml`
- never auto-create a migration evidence bundle
- never present the rationale draft as canonical migration truth
