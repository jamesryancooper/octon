# Report Packages

Multi-file report sets that do not match the bounded bundle contracts live here.

## Contract

- Use directory names that are either date-prefixed (`YYYY-MM-DD-<slug>/`) or stable topical slugs (`<slug>/`) when chronology is not the primary discovery key.
- Keep package contents self-contained and readable without depending on neighboring package directories.
- Do not use this surface for authoritative audit, migration, workflow, or decision bundles; those have dedicated roots.
