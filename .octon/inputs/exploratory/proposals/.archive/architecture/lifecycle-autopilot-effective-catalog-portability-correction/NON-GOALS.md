# Non-Goals

_Status: Draft proposal non-goals_

- Do not promote or implement Governed Workflow Runtime runtime capabilities.
- Do not change the current run lifecycle, execution authorization, context-pack,
  effect-token, evidence-store, support-target, or fail-closed contracts.
- Do not widen `program-atomic` support.
- Do not treat generated effective catalogs, generated proposal registry entries,
  proposal-local receipts, chat transcripts, agent output, MCP/tool availability,
  Durable Object state, or external workflow-engine state as authority.
- Do not make the Governed Workflow Runtime transition program closeout-ready.
- Do not create Durable Object, MCP, or external workflow-engine evaluation packets.
- Do not relax fail-closed behavior for real lifecycle-contract declaration
  defects.
