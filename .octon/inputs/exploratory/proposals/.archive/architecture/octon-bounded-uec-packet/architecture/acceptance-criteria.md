# Acceptance Criteria

`octon-bounded-uec-packet` is considered landed only when all of the
following are true:

1. The live complete-claim posture has been downgraded or superseded whenever
   blocker evidence requires recertification-open status.
2. Claim-bearing runs have coherent authority ledgers plus substantive
   instruction-layer and evidence-classification artifacts.
3. Validator-backed disclosure and release-lineage outputs align with retained
   run evidence.
4. All unresolved closure blockers in
   `traceability/01-master-closure-blocker-register.md` are either closed or
   explicitly downgraded out of the active claim envelope.
5. Two consecutive recertification passes succeed with no new material
   blocker and no claim-calibration drift.

Detailed proof burden lives in:

- `specs/03-validator-and-evidence-program.md`
- `specs/05-migration-cutover-recertification-checklists.md`
- `traceability/02-audit-finding-to-remediation-to-validator-to-evidence-matrix.md`
