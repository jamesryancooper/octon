# Deferred Scope Ledger

Deferred:

- broad effectful MCP support;
- arbitrary external API autonomy;
- browser-driving autonomy;
- deployment automation;
- credential provisioning;
- multi-repo stewardship or federation;
- autonomous support-target widening;
- autonomous governance amendments;
- fully unattended unconstrained long-horizon work;
- destructive or irreversible external operations;
- multiple simultaneous stewardship programs per repo;
- general-purpose plugin marketplace behavior;
- automatic campaign promotion without evidence-backed go decision;
- unlimited self-renewing epochs.
