# Non-Authority Surface Analysis

_Status: In-review proposal packet artifact_


| Surface | Risk | Required framing |
|---|---|---|
| Agents | Treated as control owners | Bounded/evidenced activity nodes only |
| Prompts | Treated as policy | Instruction surfaces only; not policy authority |
| Generated projections | Treated as control/evidence | Rebuildable read models only |
| Raw inputs | Treated as runtime dependencies | Non-authoritative input until promoted |
| MCP/tools | Availability treated as permission | Admitted connector operations only |
| Durable Objects | Live state treated as control truth | Future coordination adapter only |
| Browser sessions | Session state treated as evidence | Reindex into `state/evidence/**` if needed |
| External dashboards | Status treated as closeout truth | Retained Octon evidence only |
| GitHub labels/comments/issues | Host affordance treated as policy | Projection/input only |
| Chat/model memory | Treated as authority | Never authority; reindex as evidence if required |
