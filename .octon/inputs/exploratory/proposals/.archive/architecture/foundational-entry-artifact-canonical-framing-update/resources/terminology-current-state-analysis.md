# Terminology Current-State Analysis

_Status: In-review proposal packet artifact_


## Current glossary strengths

- Defines Constitutional Engineering Harness as whole system.
- Defines Governed Agent Runtime as runtime core, not whole Octon.
- Bans Model Harness as a primary term.
- Bans Framework/Bot/Assistant as primary classifications.
- Constrains Orchestrator to coordination component/role.

## Current glossary gap

It does not yet provide canonical entries for:
- Governed Workflow Runtime;
- task-specific execution harness;
- bounded agent node;
- evidenced activity node.

## Recommendation

Add those terms and clarify Governed Agent Runtime as compatibility language during transition.
