# Cutover checklist

## Before promotion
- [ ] Confirm packet scope and final dispositions
- [ ] Confirm no overlap with active UEC remediation packet that would require merge/supersession
- [ ] Confirm packet-convention drift note is understood
- [ ] Confirm maintainers agree that any promoted files land outside the proposal tree
- [ ] Confirm no recommendation violates overlay rules or class-root authority boundaries

## Additive contract phase
- [ ] Add new/extended contracts
- [ ] Add or update instance policies/workflow contracts
- [ ] Add evidence bundle shapes and sample fixtures
- [ ] Keep validators non-blocking until representative samples validate cleanly

## Control/evidence materialization phase
- [ ] Introduce run-local review disposition files
- [ ] Introduce mission-local classification files
- [ ] Introduce retained failure/distillation bundle paths
- [ ] Introduce output-envelope validation receipts

## Fail-closed enablement phase
- [ ] Enable unresolved-blocking-review failures to stop progression
- [ ] Enable proposal-required mission classes to fail closed without packet refs
- [ ] Enable output-envelope validation where adapter/runtime profiles apply

## Closure phase
- [ ] First clean validation pass recorded
- [ ] Second consecutive clean validation pass recorded
- [ ] Required retained evidence archived
- [ ] Closure certification completed
