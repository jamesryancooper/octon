# Federated Evidence and Disclosure Plan

## Principle

V4 may aggregate evidence, but aggregation never replaces source evidence.

## Connector evidence

Connector evidence must include:
- admission review evidence;
- trust dossier proof;
- validator results;
- operation receipts;
- run linkage;
- authorization grant/effect token linkage;
- rollback/compensation evidence;
- quarantine/retirement evidence when applicable.

## Generated disclosure

Generated connector catalogs, trust summaries, and support views may help operators, but they are derived-only read models.

## Future portfolio evidence

Future Stewardship Portfolio evidence can roll up connector status across repos/programs only by referencing:
- per-repo support admissions;
- per-connector dossiers;
- per-run receipts;
- per-mission evidence;
- per-release evidence.
