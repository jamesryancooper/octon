# Deferred Scope Ledger

The following are explicitly deferred:

- broad effectful MCP support;
- arbitrary external API writes;
- browser-driving autonomy;
- production deployment automation;
- credential provisioning;
- multi-repo portfolio runtime;
- cross-repo engagement runtime;
- campaign promotion runtime;
- autonomous support-target widening;
- autonomous governance amendments;
- destructive external operations;
- general-purpose plugin marketplace behavior;
- portfolio authority overriding repo-local authority.

This proposal may include structural hooks for these future capabilities but does not make them live.
