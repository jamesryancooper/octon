# Promotion Readiness Checklist

- [ ] Promotion readiness requires:
- [ ] - proposal status accepted;
- [ ] - target surfaces declared;
- [ ] - no unresolved blockers;
- [ ] - schemas validate;
- [ ] - negative controls pass;
- [ ] - generated outputs remain non-authoritative;
- [ ] - promotion receipt plan exists;
- [ ] - rollback/retirement posture exists;
- [ ] - recertification plan exists;
- [ ] - constitutional amendment route identified for elevated changes.
