# Artifact Catalog

## Policy Core

- `proposal.yml` - packet manifest and promotion targets.
- `policy-proposal.yml` - policy proposal classification.
- `policy/decision.md` - target policy decision.
- `policy/policy-delta.md` - required changes to authoritative policy
  surfaces.
- `policy/enforcement-plan.md` - validators, helper scripts, CI gates, and
  GitHub ruleset migration requirements.
- `policy/routing-model.md` - route and lifecycle vocabulary for the hybrid
  landing model.

## Implementation Planning

- `implementation/implementation-map.md` - repo-aware implementation plan by
  target artifact.
- `support/executable-implementation-prompt.md` - follow-up implementation
  prompt.
- `support/implementation-grade-completeness-review.md` - readiness receipt.
- `support/implementation-conformance-review.md` - post-implementation
  conformance scaffold.
- `support/post-implementation-drift-churn-review.md` - post-implementation
  drift/churn scaffold.

## Source Lineage

- `resources/source-context.md` - source context, live GitHub ruleset facts,
  design constraints, and proposal boundary.
