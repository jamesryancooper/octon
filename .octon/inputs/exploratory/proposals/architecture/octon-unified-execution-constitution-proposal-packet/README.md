## Non-negotiable framing

The packet preserves the already-set target-state positions:
- mission is not the atomic execution primitive; the stack is workspace charter + mission charter + run contract + execution attempt/stage
- evidence is classed across git-inline control evidence, git-tracked manifests/pointers, and external immutable replay/telemetry
- GitHub labels and host UIs are adapters, not authority
- lab is a first-class top-level domain
- portability is mediated by adapter contracts and conformance evidence
- proof is multi-plane: structural, functional, behavioral, governance, recovery
- RunCard and HarnessCard are mandatory disclosure surfaces
- browser/UI and broader API action surfaces land through governed capability packs
- support claims must be bounded by an explicit support-target matrix
- the agency layer simplifies around real boundary value, not persona inflation
