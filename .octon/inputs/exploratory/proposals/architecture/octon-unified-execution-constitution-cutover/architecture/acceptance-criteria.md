# Acceptance Criteria

## Global implementation rule
- the accepted implementation for each phase is the one that best matches the target-state architecture, not the one that merely minimizes diff size
- transitional structure is acceptable only when it is explicitly load-bearing for cutover, compatibility, or evidence and has a clear retirement path

## Phase-level exit criteria

### Phase 1 exit
- one constitutional kernel exists
- old constitutional surfaces are shims, not conflicting authorities
- ingress reads constitutional kernel first

### Phase 2 exit
- every material execution path can produce a run contract
- approvals/grants/leases/revocations exist as canonical artifacts
- host-native labels are no longer the authority source

### Phase 3 exit
- run state is resumable from artifacts
- evidence retention classes are enforced
- replay indexing works for supported run classes

### Phase 4 exit
- all proof planes exist
- structural and governance remain blocking
- at least one supported tier has behavioral and recovery evidence
- lab domain runs scenario/replay/shadow or fault interfaces in substance

### Phase 5 exit
- support-target matrix is published and enforced
- new support tiers require adapter conformance evidence
- unsupported tuples fail closed

### Phase 6 exit
- orchestrator is the kernel execution profile
- persona-heavy surfaces are optional overlays only
- at least one host-shaped authority path is deleted

### Final target-state claim criteria
Octon may claim to be a fully unified execution constitution only when:
1. constitutional kernel is supreme and singular
2. every consequential run is run-bound
3. every material action is authority-routed before effect
4. mission-backed and run-only autonomy are explicitly policy-bound
5. all required proof planes exist and are enforced for claimed support tiers
6. RunCard and HarnessCard are mandatory and sufficient disclosure surfaces
7. evidence classes and replay indexing are enforced
8. support-target matrix is real and fail-closed
9. hidden human repair is impossible without disclosure
10. transitional constitutional, persona, and host-native authority scaffolding has been retired or clearly non-authoritative
