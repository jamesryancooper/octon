## Gap interpretation rules used here

- `fully_covered` means the mechanism is already materially present in the correct canonical surfaces.
- `partially_covered` means the repo already carries adjacent machinery, but not the full closure-ready capability.
- `not_currently_present` means no materially embodied mechanism was observed, even if supporting surfaces exist.

## Immediate takeaway

The live repo is not missing the whole pattern family. It already has the most constitutionally important pieces. The remaining work is concentrated in:
- explicit structured review disposition,
- governed failure-distillation and hardening,
- explicit compact output envelopes for adapters/tools,
- explicit distillation workflow from evidence to promotable knowledge,
- and proposal-first mission classification for high-ambiguity work.
