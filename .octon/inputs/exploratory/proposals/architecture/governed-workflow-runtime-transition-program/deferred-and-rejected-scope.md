# Deferred And Rejected Scope

_Status: Draft parent-program scope register_

## Rejected As Authority

The following are rejected as Octon authority, control truth, support truth, or
closeout truth:

- Durable Object state;
- MCP server availability;
- tool availability;
- external workflow engine state;
- external dashboards;
- generated projections;
- proposal-local artifacts;
- chat transcripts;
- agent outputs;
- model memory.

## Deferred Or Lab-Only Children

### `durable-coordination-adapter-evaluation`

Durable Objects may be evaluated only as possible live coordination adapters.
They must not store canonical Octon control truth, retained evidence, authority
decisions, support claims, or closeout truth.

### `mcp-integration-evaluation`

MCP may be evaluated only as a connector/protocol input. MCP descriptors,
servers, tools, prompts, or resources must not become permission, support
admission, runtime policy, or authority.

### `external-workflow-engine-adapter-evaluation`

External workflow engines may be evaluated only as adapters, executors, or lab
targets. They must not own Octon workflow truth, run truth, closeout truth,
support claims, or authorization decisions.

## Required Mutation Before Widening

Any deferred/lab-only child can become required only through an explicit
operator-reviewed parent registry mutation with digest checks, authority-boundary
review, and rollback posture.
