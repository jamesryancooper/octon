# Init Objectives

Objective packs used by `/init` to generate Octon's repo-level objective contract:

- `.octon/OBJECTIVE.md`
- `.octon/cognition/runtime/context/intent.contract.yml`

Use `--list-objectives` to inspect the catalog, or run `/init` in an interactive terminal to choose one during initialization.

## Available Objectives

- `project-app-repo` — set up and develop a software project or application repository.
- `documentation-knowledge-base` — create, maintain, and improve docs, guides, and reference content.
- `filesystem-management` — organize, audit, migrate, and govern files and directory structures safely.
- `general-purpose` — use Octon as a governed autonomous workspace for mixed or custom objectives.
