# Validation

The skill is complete when:

- the shared wrapper runs successfully;
- both report outputs exist;
- required-path metrics are present or explicitly marked insufficient;
- top workflow and step hotspots are listed;
- duplicate-work candidates are classified;
- the final report includes a concrete next tightening action.

Reject the run as incomplete when:

- the summary JSON is missing required top-level fields;
- the report omits status or issue action;
- recommendations are generic and not tied to evidence.
