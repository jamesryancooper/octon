# Checkpoints

Record these checkpoints while planning:

1. Findings normalized
2. Release state and hard-gate facts collected
3. `change_profile` selected
4. Target charter properties declared
5. Change bundles mapped
6. Required output sections completed
7. Validation scenarios completed

If a checkpoint fails because the findings are incomplete or inconsistent, escalate that blocker instead of filling it with assumptions.
