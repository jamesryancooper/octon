# Glossary Reference

**Required when capability:** `domain-specialized`

## Core Terms

| Term | Definition | Example |
|------|------------|---------|
| MVCC | Multi-version concurrency control model in Postgres | Non-blocking reads during writes |
| EXPLAIN ANALYZE | Query plan + runtime inspection command | Diagnose slow query paths |
| index selectivity | How well an index filters rows | Choosing compound index columns |
| VACUUM | Storage cleanup and visibility maintenance process | Table bloat reduction |
| query plan | Execution strategy chosen by planner | Seq scan vs index scan decision |
