# Glossary Reference

**Required when capability:** `domain-specialized`

## Core Terms

| Term | Definition | Example |
|------|------------|---------|
| frame drop | Missed render frame causing visible jank | Animation stutter under load |
| virtualization | Rendering visible list window only | Large list performance tuning |
| gesture handler | Native-driven gesture recognition system | Pan/swipe interaction pipeline |
| Reanimated | Animation library executing work off JS thread | Smooth transitions at 60fps |
| navigation stack | Screen history model for app flow | Push/pop screen transitions |
