# Glossary Reference

**Required when capability:** `domain-specialized`

## Core Terms

| Term | Definition | Example |
|------|------------|---------|
| Expo | React Native application platform/toolchain | Managed workflow with EAS |
| JS thread | JavaScript execution thread in RN runtime | App logic and state updates |
| UI thread | Native rendering thread | Layout and frame commits |
| bridge | Messaging boundary between JS and native | Data handoff overhead |
| FlashList | High-performance list component | Large dataset virtualization |
