# Glossary Reference

**Required when capability:** `domain-specialized`

## Core Terms

| Term | Definition | Example |
|------|------------|---------|
| FastAPI | Python ASGI framework used for API endpoints | Route handlers for HTTP services |
| Pydantic | Data validation and parsing library | Strict request/response models |
| contract-first | API workflow where schema is defined before implementation | OpenAPI and JSON schema before handlers |
| Alembic | Database migration tool used with SQLAlchemy | Versioned schema migrations |
| justfile | Task runner command file | Standard dev/test/build commands |
