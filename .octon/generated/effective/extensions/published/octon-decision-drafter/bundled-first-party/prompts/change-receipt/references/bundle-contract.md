# Change Receipt Bundle Contract

This bundle drafts a concise non-authoritative markdown receipt from a diff
plus retained evidence.

## Required Outcome

- summarize the change scope and changed paths
- cite existing retained receipts or evidence
- produce one reviewable markdown receipt draft

## Additional Guardrails

- never create a new canonical receipt surface
- never auto-write under `state/evidence/**`
- never represent the draft as a substitute for retained execution or
  publication receipts
