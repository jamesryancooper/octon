# Subsystem Targeted Integration: Extract Subsystem Concepts

Extract concepts from the source artifact that are implementable inside the
declared subsystem scope.

## Shared Contracts

- apply `/.octon/inputs/additive/extensions/octon-concept-integration/prompts/shared/repository-grounding.md`
- apply the scoped subsystem record from `stages/01-scope-subsystem.md`

## Output

Produce provisional subsystem-scoped recommendations and reject concepts that
only work by escaping the declared scope without explicit justification.
