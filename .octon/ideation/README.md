Human-led exploration and idea management.

| Subdirectory | Purpose |
|--------------|---------|

## Convention Authority

- This domain has no local `practices/` surface.
- It inherits naming and authoring conventions from `/.octon/conventions.md`.

## Access Rules

- **Autonomous mode:** Treat this directory as if it does not exist.
- **Human-directed:** Access specific files only when human explicitly points to them and requests a concrete action.
- **Scope:** Stay within referenced files only. Do not browse or enumerate.

See `scratchpad/README.md` and `projects/README.md` for subdirectory details.
