# Analysis Reports

Standalone report files live here.

## Contract

- Use date-prefixed filenames:
  - `YYYY-MM-DD-<slug>.md`
  - `YYYY-MM-DD-<slug>.json`
  - `YYYY-MM-DD-<slug>.yml`
- This surface is for single-file outputs, not evidence bundle directories.
- If a deliverable grows into a multi-file report set, move it to `../packages/`.
