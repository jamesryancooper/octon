# CostKit — Token & Spend Guardrails

- **Purpose:** Budgets, per-run caps, model selection.
- **Responsibilities:** choose cheap/smart models by step; enforce limits.
- **Integrates with:** PlanKit/AgentKit (routing), ModelKit (catalog), ObservaKit (cost reports).
- **I/O:** budget usage, model decisions.
- **Wins:** Predictable bills.
