# DatasetKit — Reusable Corpora for RAG/Eval

- **Purpose:** Versioned datasets (snippets, Q/A, golden answers).
- **Responsibilities:** curate, tag, publish datasets for QueryKit/EvalKit.
- **Integrates with:** IngestKit, IndexKit, EvalKit.
- **I/O:** `datasets/**` (JSONL).
- **Wins:** Better retrieval & stable evals.
