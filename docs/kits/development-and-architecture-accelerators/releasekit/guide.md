# ReleaseKit — Release Notes & Versioning

- **Purpose:** Generate CHANGELOGs, semver bumps, GitHub releases.
- **Responsibilities:** collate PRs, extract highlights, publish.
- **Integrates with:** PatchKit, NotifyKit.
- **I/O:** `CHANGELOG.md`, releases.
- **Wins:** Clean releases with almost no effort.
- **Harmony default:** Feature-flag-aware release notes; integrate with Vercel promote/rollback flows.
