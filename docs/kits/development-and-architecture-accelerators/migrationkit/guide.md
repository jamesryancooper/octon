# MigrationKit — Schema & Data Migrations

- **Purpose:** Safe schema/data migrations with roll-forward/back strategies.
- **Responsibilities:** migration plans, backfills, online rollout checks, data validation.
- **Integrates with:** StackKit (db choices), DevKit (migration code), EvalKit (safety checks), PatchKit (migration PRs).
- **I/O:** migration scripts/reports, rollback manifests.
- **Wins:** Confident database changes without downtime.
