# BenchKit — Benchmarks & Performance Checks

- **Purpose:** Perf baselines and PR regressions.
- **Responsibilities:** micro/endpoint tests, thresholds, PR comments.
- **Integrates with:** DevKit, EvalKit, PatchKit.
- **I/O:** benchmark reports.
- **Wins:** Prevent silent perf drift.
- **Harmony default:** Tie thresholds to SLOs; burn-rate alerts via ObservaKit dashboards.
