# DiagramKit — Diagrams-as-Code

- **Purpose:** Generate/update PlantUML/Mermaid diagrams from stack/profile/graph.
- **Responsibilities:** architecture, dataflow, sequence diagrams.
- **Integrates with:** StackKit, IndexKit (graph), Dockit (docs).
- **I/O:** `.puml/.mmd/.svg` artifacts.
- **Wins:** Architecture stays current.
