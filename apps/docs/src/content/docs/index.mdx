Welcome to the Harmony documentation.

- Monorepo layout: apps and packages orchestrated with PNPM and Turbo.
- Tech stack: Astro for the marketing site, Next.js for the AI console, Node and Python services, plus shared domain/adapters/config/contracts/ui-kit packages.
- Local development: start the docs with `pnpm --filter @harmony/docs dev` or run every app via `pnpm dev:all`.

Refer to the sidebar for focused guides, architecture notes, and references.
